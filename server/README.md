# Server README

## Setup

1. Install dependencies:
```bash
npm install
```

2. Configure environment variables:
```bash
cp .env.example .env
```

Edit `.env` with your database credentials.

3. Run migrations:
```bash
npm run migrate
```

4. Seed database:
```bash
npm run seed
```

5. Start server:
```bash
npm run dev
```
